# Система volunteer-вычислений для использования свободных ресурсов

Идея:
```
Создать приложение наподобие BOINC — систему, в которой компьютеры
отправляют свой idle-ресурс для обработки задач (например, обработка
данных, вычисления).
```

Что реализовать:

- Сервер, который даёт задачи
- Клиент, который принимает и возвращает результаты
- Протокол распределения и подтверждения результатов
